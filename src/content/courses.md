# Free Courses

Anthropic runs a free learning platform, Anthropic Academy, at [anthropic.skilljar.com](https://anthropic.skilljar.com/). Each course needs only an email and gives you a completion certificate. These are genuinely good, and they pair perfectly with this guide.

Here is how to work through them depending on where you are.

## If you are brand new

Start here, in order:

1. **Claude 101** teaches you to use Claude for everyday work and its core features.
2. **Claude Code 101** gets you productive with Claude Code in your daily development.
3. **AI Fluency: Framework and Foundations** is the flagship course on collaborating with AI effectively, efficiently, ethically, and safely.

## If you want to go deep on Claude Code

- **Claude Code in Action** is a hands-on course on integrating Claude Code into your workflow.
- **Introduction to Agent Skills** shows you how to build, configure, and share Skills.
- **Introduction to Subagents** covers using and creating subagents to manage context and delegate work.
- **Introduction to Claude Cowork** walks through the task loop, plugins, skills, and file and research workflows on real projects.

## If you want to build with the API

- **Claude Platform 101** is the developer onboarding for building on the Claude Developer Platform.
- **Building with the Claude API** covers the full spectrum of working with the models programmatically.
- **Introduction to Model Context Protocol** teaches you to build MCP servers and clients in Python, using the three primitives.
- **Model Context Protocol: Advanced Topics** goes into sampling, notifications, filesystem access, and transports for production servers.

## Cloud and specialized tracks

- **Claude with Amazon Bedrock** and **Claude with Google Cloud's Vertex AI** for those platforms.
- **AI Capabilities and Limitations** for a grounded understanding of what AI can and cannot do.
- The broader **AI Fluency** series has editions for educators, students, nonprofits, small businesses, and builders.

## A suggested plan

Do not binge them. Pick one, finish it, and immediately apply it to a real task in your own project the same day. That is how the knowledge sticks. A course plus one real use equals a skill you actually own.

The catalog grows over time, so browse [the full list](https://anthropic.skilljar.com/) for anything new.

Prefer learning in code? The [anthropics/courses](https://github.com/anthropics/courses) notebooks on GitHub walk through API fundamentals, prompt engineering, real-world prompting, prompt evaluations, and tool use, all as runnable Jupyter notebooks.

**Also great:** [anthropic.com/learn](https://www.anthropic.com/learn) is the learning hub, and the [links page](/docs/links) here collects every official resource in one spot.
